# デザイントークン（配色・タイポグラフィ・余白）

ポートフォリオサイト（my-portfolio2026）の配色・タイポグラフィ・余白の具体値をまとめたドキュメント。

- 関連 Issue: [#4 デザイントークン（配色・タイポグラフィ・余白）を定める](https://github.com/kainholy/my-portfolio2026/issues/4)
- 方針・トンマナ: [design-concept.md](./design-concept.md) / [#2 トンマナとデザインコンセプト](https://github.com/kainholy/my-portfolio2026/issues/2)

色を持つのは実績画像のみ。地とテキストは温かみのある無彩色に寄せ、アクセントは彩度を抑えたアースカラー1色のみとする。

---

## 配色

| 用途 | 値 | 補足 |
| --- | --- | --- |
| 地 / 背景（生成り） | `#F6F2E9` | 基調。純白は使わない |
| 背景・一段（区切り） | `#EFEADD` | セクションの切り替えに薄く |
| カード / 画像の余白地 | `#FBF8F1` | モックアップの下地 |
| 文字・主 | `#2A2724` | 純黒を避けた温かみのあるダークグレー |
| 文字・副 | `#6B655C` | リード・補足 |
| 文字・メタ | `#9A9488` | キャプション・使用技術など |
| 罫線 / ボーダー | `#E0D9CA` | ヘアライン（0.5px） |
| アクセント（1色・最小限） | `#7A7150` | くすんだオリーブ系。リンク hover / 区切りの点など要所のみ |

## タイポグラフィ

和文は明朝（`Noto Serif JP` 300/400/500）、欧文ラベルは細めのセリフ（`Cormorant Garamond`）またはサンセリフ（`Inter`）。文字は小さめ・字間広め・行間ゆったり。

| 用途 | PC | SP | ウェイト / 字間・行間 |
| --- | --- | --- | --- |
| FV キャッチ | 40px | 28px | 明朝 400 / letter-spacing .08em |
| セクション欧文ラベル | 13px | 12px | .28em・大文字 |
| 和文見出し | 23px | 20px | 明朝 500 / .06em |
| WORKS 案件タイトル | 20px | 18px | 明朝 400 |
| 本文 | 15px | 14px | line-height 1.95 / .06em |
| メタ・キャプション | 12px | 12px | サンセリフ / .10em |

- 共通: `body` の `letter-spacing: .06em`、本文行間 1.9〜2.0、`font-feature-settings: "palt"`

## レイアウト・余白

引き算の設計。要素間を広く取り、スクロールで一つずつ見せる。

| 項目 | PC（1440基準） | SP（390基準） |
| --- | --- | --- |
| コンテンツ最大幅 | 1040px（中央寄せ） | 100% |
| 左右ガター | 80px（`clamp(24px, 6vw, 80px)`） | 24px |
| セクション間の縦余白 | 180px | 100px |
| 見出し → 本文 | 32px | 24px |
| 段落間 | 24px | 20px |
| WORKS 各件の間隔 | 120px | 72px |

## 要検討

- アクセント色: オリーブ `#7A7150` を基本とする。テラコッタ系（`#A5714E`）等の候補も検討
- 欧文フォント: ラベルをセリフ（Cormorant）で上品に寄せるか、サンセリフ（Inter）で端正に締めるか
